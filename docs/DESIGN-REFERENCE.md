# Command Glass visual reference

The repository follows the approved Command Glass direction:

- Canvas: `#090A0B`
- Graphite: `#131415`
- Brand gold: `#F2B900`
- Warm white: `#F5F2EA`
- Silver: `#B4B1AA`
- Header: smoked glass with a restrained gold bottom hairline
- Hero: short text-led presentation with **Choose. Play. Swap. Repeat.**
- Desktop catalog: 300 px filter rail plus horizontal game cards
- Every card: cover, title, platform/category badges, separate Trophy and Non-Trophy rows
- Available: Messenger action
- Not available: waitlist action
- Tablet/phone: filter bottom sheet and horizontal one-card-per-row layout
