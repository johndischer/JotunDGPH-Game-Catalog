# Game covers

Upload production covers in this folder.

Recommended:

- 1024 × 1024 pixels
- `.webp` preferred; `.jpg` supported
- ideally below 250 KB
- exact filename match with **Game Details → Cover Filename**

Example:

```text
public/covers/JG-0029.webp
```

Sheet value:

```text
JG-0029.webp
```

The included SVG files are preview placeholders and may be deleted after live covers are uploaded.
